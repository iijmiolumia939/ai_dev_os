# Session Mode Routing

Session Mode Router combines session focus, topic isolation, continuity scope, repository subset, architecture hotspots, and validation status.

## Output

- Recommended mode.
- Fallback mode.
- Escalation requirement.
- Compact mode.
- Isolation requirement.
- Recommended prompt type.

The fallback mode is always bounded. Routing can recommend isolation, but it does not automatically expand context or escalate reasoning beyond the selected profile budget.
