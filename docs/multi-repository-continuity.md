# Multi Repository Continuity

Multi Repository Continuity maps AI_DEV_OS platform and consumer repositories without exporting repository contents.

## Summary Fields

- Repository relationships.
- AI_DEV_OS consumer repositories.
- Isolated repositories.
- Stale repositories.
- Migration state.
- Adapter compatibility state.
- Governance compatibility.
- Dependency graph summary.

The dependency graph is intentionally compact. It reports node and edge counts plus bounded relationship labels instead of indexing code or exporting trees.
