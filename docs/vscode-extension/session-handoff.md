# VSCode Session Handoff

Session handoff is local and human-confirmed.

## Workflow

1. Run session audit.
2. Generate a compact continuity bundle.
3. Review stale session and rollover warnings.
4. Copy the prompt or export the compact handoff.
5. Start the new session manually.
6. Confirm rollover only after the human has reviewed the handoff.

The extension does not automate chat UI, Copilot Chat, browser actions, or consumer repository mutation.
