# Reasoning Profile Governance

Reasoning Profile Governance maps session focus into bounded profiles:

- bounded_implementation
- isolated_architecture
- rollout_review
- governance_review
- debugging
- retrieval_maintenance
- provider_analysis
- renderer_review

Each profile defines reasoning depth, architecture allowance, retrieval budget, review intensity, continuity preservation, and escalation policy. Profiles do not permit unrestricted reasoning escalation.
